# MERN_CRUD_App

This is a Learning Express Server & MongoDB Configuration with REST API Creation & Testing with Postman.


## Back End
* Express Server
* MongoDB
* Postman 

## Back End packages
* `npm i express nodemon cors mongoose body-parser`
* `npm i concurrently`


## Front End
* React JS
* Boostrap 5 

## Front End packages
* `npx create-react-app client`
* `npm i axios`
* `npm i react-router-dom`
